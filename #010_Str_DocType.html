HTML 文档类型声明

Web 世界中存在许多不同的文档。HTML 也有多个不同的版本。

所以，怎样才能让浏览器明白，现在浏览的这个HTML文档是什么版本？什么类型呢？

这就要用 文档类型声明 !doctype

常用的声明

  • [!doctype html]——HTML 5
  • [!DOCTYPE HTML PUBLIC "-//...//DTD HTML 4.01 Transitional//EN" "http://www....../TR/html4/loose.dtd"]——HTML 4.01
  • [!DOCTYPE html PUBLIC "-//...//DTD XHTML 1.0 Transitional//EN" "http://www....../TR/xhtml1/DTD/xhtml1-transitional.dtd"]——XHTML 1.0

不过我相信，大多数人是不想用 HTML 5 以外的声明的XD
